# Estética Delicado - Landing Page

Uma landing page charmosa e moderna para o salão de estética Estética Delicado, especializada em alongamento de cílios e unhas em gel.

## ✨ Características

- **Design Hello Kitty**: Estética fofa e delicada com cores rosa e lilás
- **Serviços Específicos**: Alongamento de cílios (Brasileiro, Russo, Fox, Natural) e unhas em gel
- **Responsivo**: Funciona perfeitamente em desktop e mobile
- **Navegação Suave**: Rolagem suave entre seções
- **Botões Interativos**: Links diretos para WhatsApp e Instagram

## 🚀 Tecnologias Utilizadas

- HTML5
- CSS3 (com variáveis CSS e animações)
- Design responsivo com media queries

## 📱 Seções

1. **Hero**: Apresentação e call-to-action principal
2. **Serviços de Cílios**: Detalhes dos tipos de alongamento
3. **Serviços de Unhas**: Alongamento em gel
4. **Sobre**: Informações sobre o salão
5. **Agendamento**: Botões para WhatsApp e Instagram
6. **Contato**: Informações de contato no footer

Para personalizar:
- Altere os links do WhatsApp e Instagram nos botões
- Modifique as cores nas variáveis CSS
- Atualize as informações de contato
- Adicione suas próprias imagens se desejar
